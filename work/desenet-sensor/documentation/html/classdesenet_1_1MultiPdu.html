Detailed Description

Object to represent a Multi PDU Frame.

The class can be used to construct a Multi PDU frame to send or to decompose the different PDUs of a received Multi PDU frame.

◆ sampledValuesInsertionBuffer()

desenet::SharedByteBuffer desenet::MultiPdu::sampledValuesInsertionBuffer ( )

Returns a shared buffer that can be used to write sampled values data directly into the Multi-PDU frame. This avoids unnecessary copies.

It is important to call the method commitSampledValues() after having written to the shared buffer in order to add the metadata to the embedded PDU, otherwise the written data will be ignored. NEVER store a copy of this buffer when manipulating an Multi-PDU frame, as the shared buffer returned by this method will change with the progress of the construction of the frame.

Returns
Shared buffer to write the data to.
