Detailed Description

Frame send or received by the lower layer.

This class may be use to construct more specific frame types used by the higher layers.

void desenet::Frame::setLength ( std::size_t length)
protected

Sets the frame length. Includes header, payload and footer of the frame.

Because the buffer backing the frame object can not be changed in size, the length of the frame logically can not exceed. The maximal possible frame size can be received using the method reservedLength().

Parameters
lengthThe new length of the frame.
